# 🛒 DigiTools — Digital Tools Buying Platform

> A modern, fully responsive digital marketplace where users can browse, add to cart, and purchase premium digital tools — built with React and styled with Tailwind CSS + DaisyUI.

## ✨ Features

### 🛒 Smart Cart System

Add products to your cart with a single click, view a live cart count in the navbar, remove individual items, and proceed to checkout — which clears the entire cart instantly.

### 🔔 Real-Time Toast Notifications

Every cart action — adding a product, removing an item, or completing checkout — triggers a clean, non-intrusive toast notification powered by React-Toastify.

### 📱 Fully Responsive Design

Optimized for all screen sizes from mobile to desktop. The product grid adapts from 1 to 3 columns, and every section — navbar, banner, pricing, footer — scales gracefully across devices.

## 📦 Getting Started

```bash
# Clone the repository
git clone https://github.com/saidurrahmanratul/digitools.git

# Navigate into the project
cd digitools

# Install dependencies
npm install

# Start the development server
npm run dev
```
